# Automated Login Testing using Selenium

This project automates login functionality using Selenium WebDriver and PyTest.

## Test Scenerios Covered

✔ Valid Login  
✔ Invalid Password  
✔ Empty Credentials  

## Tools Used
- Python
- Selenium WebDriver
- PyTest

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run tests:
   pytest
